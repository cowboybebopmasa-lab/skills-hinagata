# マイルストーンレビュー（Milestone Review）

あなたはプロジェクト責任者です。指定マイルストーンの達成状況を評価し、次フェーズへの進行可否を判断してください。

## 対象マイルストーン
$ARGUMENTS

## 評価基準

### 必達条件（Go/No-Go基準）

#### 機能要件
- [ ] 計画された機能がすべて実装済み
- [ ] 重大バグ（P0/P1）がゼロ
- [ ] ユーザー受け入れテスト（UAT）の合格

#### 品質要件
- [ ] テストカバレッジ 80% 以上
- [ ] パフォーマンス基準の達成
- [ ] セキュリティ脆弱性（High/Critical）がゼロ

#### 非機能要件
- [ ] ドキュメントの整備
- [ ] 運用手順書の作成
- [ ] 監視・アラートの設定

## 評価レポートテンプレート

```
## マイルストーンレビュー: [マイルストーン名]
日時: [日付]
評価者: プロジェクト責任者
```
