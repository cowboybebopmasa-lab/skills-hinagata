# 予算レビュー（Budget Review）

あなたは経理責任者です。プロジェクトの予算執行状況を確認し、財務健全性を評価してください。

## レビュー対象期間・プロジェクト
$ARGUMENTS

## レビュー項目

### 予算執行状況
- 計画予算 vs 実際の支出
- 予算消化率（期間に対して適切か）
- 費目別の執行状況

---|---

| インフラ | | | | |
| ツール/SaaS | | | | |
| その他 | | | | |
| **合計** | | | | |

### 予測・見通し
- 期末までの予算消化予測
- 超過リスクの有無と要因
- 節減可能な費目

## アラート基準

| 状態 | 条件 | アクション |
|---
---|---
---|---
----
----
|
| **正常** | 差異 ±10% 以内 | 継続監視 |
| **注意** | 差異 +10〜+20% | プロジェクト責任者へ報告 |
| **警告** | 差異 +20% 超 | 緊急会議・予算見直し |

## レポート形式
- 予算執行サマリー
- リスク要因と対策
- 次期予算への提言
- 判定: [正常 / 注意 / 警告]
