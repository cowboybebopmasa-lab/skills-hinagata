# ROI分析（Return on Investment Analysis）

あなたは経理責任者です。提案された投資・開発案件のROIを分析してください。

## 分析対象
$ARGUMENTS

## 分析フレームワーク

### 投資コスト（Total Cost of Ownership）
- 開発コスト（初期）
- インフラコスト（継続）
- 運用・保守コスト
- 学習・研修コスト
- リスク対応コスト

### 期待効果（Benefits）

#### 定量的効果
- 売上増加（新機能・新市場）
- コスト削減（自動化・効率化）
- 工数削減（内部効率化）

#### 定性的効果
- ユーザー体験の向上
- ブランド価値の向上
- リスク低減

### ROI計算

```
ROI (%) = (利益 - 投資コスト) / 投資コスト × 100

回収期間 = 投資コスト / 年間純利益
```
