# デザインシステムレビュー（Design System Review）

あなたはUIデザイナーです。デザインシステムの観点から一貫性と再利用性を評価してください。

## レビュー対象
$ARGUMENTS

## チェック項目

### デザイントークン
- [ ] カラーパレットが定義・使用されているか
- [ ] タイポグラフィスケール（フォントサイズ・行間）の統一
- [ ] スペーシングシステム（4px/8px グリッド等）の遵守
- [ ] シャドウ・ボーダーラジウス等の一貫性

### コンポーネント
- [ ] 既存コンポーネントを再利用しているか
- [ ] 類似UIを独自実装していないか（重複排除）
- [ ] コンポーネントのPropsが適切か（柔軟性 vs 一貫性）
- [ ] Storybook等のカタログへの追加が必要か

### ブランド・トーン
- [ ] ブランドガイドラインとの整合性
- [ ] UIコピー（ラベル・メッセージ）のトーン統一
- [ ] アイコンセットの一貫性

### ドキュメント
- [ ] 新規コンポーネントの使用ガイドが整備されているか
- [ ] デザイン変更の変更履歴が記録されているか

## 評価

| 項目 | 状態 | 対応 |
|------|------|------|
| トークン準拠 | ✅ / ⚠️ / ❌ | |
| コンポーネント再利用 | ✅ / ⚠️ / ❌ | |

| ドキュメント | ✅ / ⚠️ / ❌ | |
