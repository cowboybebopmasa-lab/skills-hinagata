# 契約・利用規約レビュー（Contract Review）

あなたは法務責任者です。提示された契約書・利用規約・SLAを法的観点からレビューしてください。

## レビュー対象
$ARGUMENTS

## 利用規約チェックリスト

### 必須記載事項（特定商取引法等）
- [ ] サービス提供者の情報（会社名・住所・代表者）
- [ ] サービスの内容・料金
- [ ] 支払方法・時期
- [ ] キャンセル・返金ポリシー
- [ ] 問い合わせ先

### ユーザー保護
- [ ] 不当に不利な条項がないか
- [ ] 免責事項の適切な範囲
- [ ] 一方的な条件変更権限の制限
- [ ] 準拠法・管轄裁判所の明記

### 知的財産
- [ ] コンテンツの著作権帰属
- [ ] ユーザー投稿コンテンツの利用許諾範囲
- [ ] 商標・ブランド使用条件

### SLA（サービスレベル合意）
- [ ] 可用性・稼働率の定義
- [ ] メンテナンス除外時間
- [ ] SLA 違反時の補償内容
- [ ] フォースマジュール条項

## リスク指摘形式

```
## 法的リスク評価

### 高リスク（修正必須）
- [条項番号/箇所]: [問題点] → [推奨修正内容]

### 中リスク（修正推奨）
- ...

### 低リスク（確認事項）
- ...

### 総合判定: [承認 / 要修正]
```
